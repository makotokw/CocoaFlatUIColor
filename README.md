WZYFlatUIColor
========

## Installation

by CocoaPods

```ruby
pod 'WZYFlatUIColor', :git => 'https://github.com/makotokw/CocoaWZYFlatUIColor.git'
```

## Usage

```
#import "WZYFlatUIColor.h"
// ...
textLabel.textColor = [UIColor wzy_sunFlowerFlatColor];
```
