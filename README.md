CocoaFlatUIColor
========

## Installation

by CocoaPods

```ruby:Podfile
pod 'FlatUIColor', :git => 'https://github.com/makotokw/CocoaFlatUIColor.git'
```

## Usage


```
#import <FlatUIColor/UIColor+FlatUI.h>
```
